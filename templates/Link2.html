常用 HTML 標記

<h1> 標籤用來指定一級/最重要的標題。 <p> 標籤用來包含一段文字段落。

開新HTML文檔時常做的宣告

<!DOCTYPE html> <html lang="en"></html> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title> Document </title> </head> <body> </body> </html>

標籤說明

<!DOCTYPE html> 標籤用來聲明這是一份採用 HTML5 語法標準的文件。 <html> 標籤是 HTML 文件的根元素。

  裡面的 lang="語言" 目的是讓瀏覽器識別網頁的語言

  lang="en" 英文

  lang="zh-Hant" 繁體中文

<head> 標籤中的包含著不會被顯示在頁面上的內容，用來說明關於該網頁的元資訊 (metadata)。 <title> 標籤用來指定該網頁的標題，通常會顯示在瀏覽器的頁籤上。 <body> 標籤中包含著會被顯示在頁面上的內容。

常用小技巧

一、使用 &emsp; 可以出現類似 Tab 鍵的一段空白的效果。

二、如何在 HTML 文檔中顯示 HTML 標記符號呢？

答案為，使用標籤<xmp>各種符號、空格、原始碼</xmp>就能顯示所有內容符號，包含各種空格。
